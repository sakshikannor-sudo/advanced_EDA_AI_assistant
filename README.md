🚀 InsightFlow: Advanced EDA & AI Assistant
<p align="center"> <img src="assets/insightflow_banner.gif" alt="InsightFlow Demo" width="900"/> </p>

📊 An AI-powered Exploratory Data Analysis (EDA) app built using Google AI Studio that enables users to upload datasets, visualize insights, and chat with an AI assistant for smart analysis.

✨ Features

📁 Upload CSV dataset

📊 Interactive visualizations (Bar, Line, Scatter, Pie)

🧠 AI-powered data insights (Gemini Assistant)

📉 Missing value & column analysis dashboard

🔍 Smart search & filtering

💬 Chat with dataset using AI

📦 Export processed data
